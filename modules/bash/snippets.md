# Code Snippets for Bash-Scripting

### get current year
`date +%Y`

### loop through range
```bash
for i in 1 2 3 
do
  echo $i
done
```

### create directory-tree with one command
`mkdir -p root/child/sub`

### list filesystem-structure as tree
`tree /var/log`
